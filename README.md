# Complexity Guard

Always-on engineering guardrail for reducing overengineering, preserving local codebase patterns, and running simplify sweeps.

## Usage

Ask for an overengineering, complexity, maintainability, or simplify review. The skill also acts as a standing constraint when writing or reviewing code.

## Installation

```bash
ln -s ~/Developer/Personal/skill-complexity-guard/skills/complexity-guard ~/.claude/skills/complexity-guard
```
